# Prism

## Overview

Prism is websocket server based on EventMachine.

## Usage

  # start Prism
  $ cd [prism_dir]
  $ script/daemon start -- production

  # Prism status
  $ cd [prism_dir]
  $ script/daemon status      #=> runner: running [pid 2273]

  # stop prism
  $ cd [prism_dir]
  $ script/daemon stop
